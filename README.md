Oxente
======

Tradução do módulo `this` do Python para PT-BR-NE (Português brasileiro nordestino)

**Bote quente, num seja amarrado. Teje livre pra traduzir mais Eu!**

Instalar
=======
```sh
$ pip install oxente
```

Executar
=======
```py
>>> import oxente
```
Ou no modo hacker:
```sh
$ python -m oxente
```
